# Emoji Kitchen Keyboard Support Web API repo

## Setup
- Permission
  ```sh
  export AZURE_SUBSCRIPTION_ID='<subscriptionId>'
  export AZURE_TENANT_ID='<tenantId>'
  export AZURE_CLIENT_ID='<servicePrincipalId>'
  export AZURE_CLIENT_SECRET='<password>'
  ```
- Dependencies
  ```sh
  npm install
  ```
